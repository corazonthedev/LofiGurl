# Changelog

All notable changes to LofiGurl are documented here.

---

## [1.0.0] — 2026-03-12

### Added
- Initial release
- Live lofi girl 24/7 radio stream via yt-dlp + ffmpeg
- Custom frameless dark UI with hand-drawn lofi girl illustration
- Custom titlebar with minimize (`─`) and close (`✕`) buttons
- Volume bar with click-to-set and `+` / `−` buttons
- Play / pause toggle
- Session timer
- Always-on-top window
- Draggable window
- System tray minimize via pystray (hidden icons area)
- Lofi girl `.ico` icon — hand-drawn, multi-size (16–256px)
- "made by corazonthedev" footer link → github.com/corazonthedev
- No console window (`CREATE_NO_WINDOW` flag on all subprocesses)
- Auto-installs missing Python dependencies on first run
